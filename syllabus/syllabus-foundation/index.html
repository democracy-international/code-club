DI Code Club Topics (Foundation)

Part I: The Modern Web

Week One: HTML and CSS Basics

  1. What is the web and how do websites work?
  2. What do we want to learn?
  3. Assignment: “About You” on Codeacademy

Week Two: Developing Static Sites

  1. Basic Development Setup
  2. Chrome Developer Tools
  3. Bootstrap and other “Front End User Interface Frameworks”
  4. Assignment: Codecademy HTML and CSS Track

Week Three: Intro to Mapping

  1. Introduction to TileMill/CartoCSS
  2. Walk-through of building a basic map
  3. Some data sources to get you started
  4. Assignment: Make your first map on anything you like. Must include at least three layers.

Week Four: Advanced TileMill

  1. Mapping Data
  2. Common TileMill Issues
  3. Putting Maps Online
  4. Assignment: Add some data to your map and put it online

Week Five: Build Something

  1. What do we want to build?
  2. Assignment: Build it (in groups)!

Part II: The Interactive Web

Week Six: Intro to JavaScript and jQuery

  1. What is JavaScript?
  2. JS Concepts and Topics (variables, functions, objects, libraries)
  3. jQuery – The King of JS Libraries
  4. Assignment: “Animate your name” on Codeacademy

Week Seven: Digging into jQuery

  1. What can jQuery do for you? (selections, manipulating HTML and CSS, events, etc.)
  2. jQuery Tools and Plugins (jQuery UI, etc., etc.)
  3. Assignment: Codecademy jQuery Track

Week Eight: Awesome JS Libraries and Resources

  1. User Interaction: Bootstrap JS Components
  2. Charts and Graphs: D3 and HighCharts
  3. Mapping: Mapbox.js and Leaflet
  4. How to keep up on JS happenings? (JS Weekly, Dashing D3, 5 Minutes of JS Podcast, etc.)
  5. Assignment: Add some JS to your project from Week 5

Week Nine: Collaborative Web Development (Github)

  1. What Is GitHub and Why Is It Awesome?
  2. Learning the GitHub process and workflow
  3. Assignment: Create a repository for your project from Week 5 and add a new feature using the GitHub workflow

Part 3: What Else Do We Want to Learn?

  • Data science and visualization for the web
  • Advanced user interaction (UI)/user experience (UX) topics
  • From web pages to web apps (building complex sites that allow for lots of user interaction)
  • Basics of good web design (just because you can build it doesn’t mean it’ll look good)
  • Others?
